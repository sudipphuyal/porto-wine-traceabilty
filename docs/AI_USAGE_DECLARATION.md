# Declaração de utilização de IA

Este trabalho contou com apoio de ferramentas de inteligência artificial para brainstorming, explicação de código, estruturação de documentação e apoio à depuração.

O estudante declara que reviu o código, a documentação e os testes antes da submissão, compreendeu o funcionamento do modelo DAML/Canton apresentado e validou localmente os comandos indicados quando as ferramentas necessárias estavam disponíveis.

Esta declaração não afirma que a IA executou testes que não foram executados. No ambiente em que este repositório foi preparado, `daml`, `dpm` e `canton` não estavam disponíveis, pelo que os resultados runtime devem ser obtidos e anexados pelo estudante após instalação.
